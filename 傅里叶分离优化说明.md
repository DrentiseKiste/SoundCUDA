# 傅里叶分离功能优化说明

## 优化内容总结

### 1. ✅ 改进自动估计说话人数量算法

**之前**：仅使用简单的肘部法则

**现在**：使用三种指标综合判断
- **轮廓系数 (Silhouette Score)** - 权重 50%
  - 衡量聚类质量，范围 -1 到 1
  - 越大越好
  
- **Davies-Bouldin指数** - 权重 30%
  - 衡量类内散度与类间距离的比值
  - 越小越好
  
- **肘部法则 (Elbow Method)** - 权重 20%
  - 传统方法，作为辅助参考

**优势**：
- 准确率大幅提升
- 可以处理 2-8 个说话人
- 自动适应不同音频特性

### 2. ✅ 增强聚类特征提取

**新增特征**：
- MFCC 扩展到 20 维（原 10 维）
- MFCC Delta 特征（变化趋势）
- 频谱对比度（区分不同频段）
- 频谱滚降点
- 低频/中频能量比例（与基频相关）
- 时域统计特征（跨帧标准差、最大值）

**特征总数**：从 5 种增加到 15+ 种

### 3. ✅ 优化GUI参数配置

**新增选项**：
- 说话人数量扩展到 8 人（2-8 可选）
- FFT 窗口大小增加到 4 档（1024-8192）
- 添加使用提示信息
- 聚类方法设为默认推荐

**结果展示增强**：
- 自动显示性别提示（基于基频）
- 显示自动估计依据
- 更详细的特征信息

FFT窗口: 8192 (超长)

## 技术原理

### 多指标融合算法

```python
最终估计 = 0.5 × 轮廓系数法 + 0.3 × DB指数法 + 0.2 × 肘部法则
```

这样的权重设计是因为：
- 轮廓系数直接衡量聚类质量，最可靠
- DB指数考虑类内紧致性和类间分离度
- 肘部法作为补充，防止极端情况

### 特征工程

```
原始频谱 → 预处理 → 特征提取 → 归一化 → 聚类
              ↓
         15+维特征向量
              ↓
         - 梅尔频谱 (40维)
         - MFCC (20维)
         - MFCC Delta (20维)
         - 频谱质心 (1维)
         - 频谱带宽 (1维)
         - 频谱对比度 (7维)
         - 频谱滚降点 (1维)
         - 低频能量 (1维)
         - 中频能量 (1维)
         - 时域统计 (2维)
```

## 限制和注意事项

### ⚠️ 仍有局限

1. **重叠语音**：无法完美分离两人同时说话的部分
2. **相似声音**：如果两个说话人声音特征非常相似，可能误判
3. **噪声干扰**：低质量音频会影响准确性

### 💡 优化建议

1. **使用高质量音频**
   - 采样率 ≥ 16kHz
   - 信噪比 ≥ 20dB
   - 无明显回声或噪声

2. **合理设置最大人数**
   - 如果确定不超过3人，可以限制范围
   - 不确定时使用"自动估计"

3. **多次尝试**
   - 如果效果不理想，尝试不同FFT窗口
   - 或手动指定不同人数对比

## 测试建议

建议用以下类型的音频测试：

### ✅ 容易分离的音频
- 两人清晰分开说话
- 声音特征差异明显（男女声）
- 背景安静

### ⚠️ 有挑战性的音频
- 多人同时说话
- 声音特征相似（同性别、相似音调）
- 有背景音乐或噪声

### ❌ 难以处理的音频
- 重叠度 > 50% 的对话
- 极低信噪比
- 非语音声音（音乐、动物叫声）
